## 目標

第一版維持純前端，優先把專案展示、筆記管理與內容閱讀體驗整理好，讓更新和維護都保持輕量。

## MVP 頁面

- 首頁：個人定位、精選專案、最新筆記
- 專案：作品卡片與後續深度紀錄入口
- 筆記：Markdown 清單、搜尋與標籤
- 關於：簡短介紹與學習方向

## 前端先行

後端功能可以晚一點，先確保內容結構、閱讀流程與頁面視覺都已經穩定。

```js
const nextStep = '以真實專案內容取代示範資料，持續更新每週學習紀錄'
```

```ts file="content-map.ts"
export const sections = ['首頁', '專案', '筆記', '關於']
```
