## 專案定位

這裡是長期維護的學習型知識庫，目的不是一次性筆記，而是可回查、可延伸、可重用的技術資料。

## 你會在這裡看到的內容

- Agent / MCP / Tool Calling 相關學習紀錄
- LLM、RAG、Transformer 等核心概念整理
- LangChain / LlamaIndex 實務導向筆記
- Git、Pytest、TypeScript 等工程工具補充

## 閱讀建議

可依你目前需求選擇路線：

1. 想補基礎：`LLM Course`、`使用 TRANSFORMERS`、`微調一個預訓練模型`
2. 想做實務：`MCP`、`RAG`、`Tool Calling` 系列
3. 想補工程能力：`Git`、`Pytest`、`TypeScript` 系列
